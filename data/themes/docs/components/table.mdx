## API 参考

### Root

将 `Header` 和 `Body` 部分分组。该组件基于 `table` 元素，并支持 [常见的边距属性](/themes/docs/overview/layout#margin-props)。

<ThemesPropsTable defs="tableRootPropDefs" />

### Header

包含表格的列标题，基于 `thead` 元素。

### Body

显示表格数据。该组件基于 `tbody` 元素。

### Row

一行表格单元格。基于 `tr` 元素。

<ThemesPropsTable defs="tableRowPropDefs" />

### Cell

一个基本的表格单元格。该组件基于 `td` 元素，但使用 `justify` 而不是 `align` 来控制表格单元格内的水平空间分配。

<ThemesPropsTable defs="tableCellPropDefs" />

### ColumnHeaderCell

表格列的标题。基于 `th` 元素，并提供与 `Cell` 部分相同的属性接口。

<ThemesPropsTable defs="tableCellPropDefs" />

### RowHeaderCell

表格行的标题。基于 `th` 元素，并提供与 `Cell` 部分相同的属性接口。

<ThemesPropsTable defs="tableCellPropDefs" />
