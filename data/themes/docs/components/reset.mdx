## API 参考

重置组件用于在特定元素上强制重置浏览器样式。

它渲染一个 [Slot 原始元素](/primitives/docs/utilities/slot)，如下：

- 接受一个 React 元素作为其子元素
- 移除特定的浏览器样式
- 设置惯用的布局默认值，例如图像的 `display: block` 或输入框的 `width: stretch`
- 根据您的主题设置设置 [cursor](/themes/docs/theme/cursors) 样式
- 添加 `box-sizing: border-box`
