## API 参考

此组件继承自 [单选组原语](/primitives/docs/components/radio-group) 的属性，并支持 [常见的边距属性](/themes/docs/overview/layout#margin-props)。

<ThemesPropsTable defs="radioCardsRootPropDefs" />

### 项目

组中的一个项目，可以被选中。

## 示例

### 尺寸

使用 `size` 属性来控制尺寸。

```jsx live=true
<Flex align="center" gap="3">
	<RadioCards.Root size="1">
		<RadioCards.Item value="1">8核CPU</RadioCards.Item>
	</RadioCards.Root>

	<RadioCards.Root size="2">
		<RadioCards.Item value="1">8核CPU</RadioCards.Item>
	</RadioCards.Root>

	<RadioCards.Root size="3">
		<RadioCards.Item value="1">8核CPU</RadioCards.Item>
	</RadioCards.Root>
</Flex>
```

### 变体

使用 `variant` 属性来控制视觉样式。

```jsx live=true
<Flex direction="column" gap="3" maxWidth="200px">
	<RadioCards.Root variant="surface">
		<RadioCards.Item value="1">8核CPU</RadioCards.Item>
	</RadioCards.Root>

	<RadioCards.Root variant="classic">
		<RadioCards.Item value="1">8核CPU</RadioCards.Item>
	</RadioCards.Root>
</Flex>
```

### 颜色

使用 `color` 属性来指定特定的 [颜色](/themes/docs/theme/color)。

```jsx live=true
<Flex direction="column" gap="3" maxWidth="200px">
	<RadioCards.Root defaultValue="1" color="indigo">
		<RadioCards.Item value="1">8核CPU</RadioCards.Item>
	</RadioCards.Root>

	<RadioCards.Root defaultValue="1" color="cyan">
		<RadioCards.Item value="1">8核CPU</RadioCards.Item>
	</RadioCards.Root>

	<RadioCards.Root defaultValue="1" color="orange">
		<RadioCards.Item value="1">8核CPU</RadioCards.Item>
	</RadioCards.Root>

	<RadioCards.Root defaultValue="1" color="crimson">
		<RadioCards.Item value="1">8核CPU</RadioCards.Item>
	</RadioCards.Root>
</Flex>
```

### 禁用

```jsx live=true
<Flex direction="column" gap="4" maxWidth="450px">
	<RadioCards.Root columns="2" defaultValue="2">
		<RadioCards.Item value="1">关闭</RadioCards.Item>
		<RadioCards.Item value="2">开启</RadioCards.Item>
	</RadioCards.Root>

	<RadioCards.Root columns="2" defaultValue="2">
		<RadioCards.Item value="1" disabled>
			关闭
		</RadioCards.Item>
		<RadioCards.Item value="2" disabled>
			开启
		</RadioCards.Item>
	</RadioCards.Root>
</Flex>
```
