## API 参考

### Root

包含选择器的所有部分。它从选择基础部分 [Root](/primitives/docs/components/select#root) 继承属性。

<ThemesPropsTable defs="selectRootPropDefs" />

### Trigger

用于切换选择器的按钮。该组件从选择基础部分 [Trigger](/primitives/docs/components/select#trigger) 和 [Value](/primitives/docs/components/select#value) 继承属性。它支持 [常见的边距属性](/themes/docs/overview/layout#margin-props)。

<ThemesPropsTable defs="selectTriggerPropDefs" />

### Content

当选择器打开时弹出的组件。它从 [Select.Portal primitive](/primitives/docs/components/select#portal) 和 [Select.Content primitive](/primitives/docs/components/select#content) 继承属性。

<ThemesPropsTable defs="selectContentPropDefs" />

### Item

包含选择项的组件。它从 [Select.Item primitive](/primitives/docs/components/select#item) 部分继承属性。

### Group

用于分组多个项。它从 [Select.Group primitive](/primitives/docs/components/select#group) 部分继承属性。与 `Select.Label` 一起使用，以确保通过自动标签实现良好的可访问性。

### Label

用于渲染组的标签，无法通过方向键聚焦。它从 [Select.Label primitive](/primitives/docs/components/select#label) 部分继承属性。

### Separator

用于在选择器中视觉上分隔项。它从 [Select.Separator primitive](/primitives/docs/components/select#separator) 部分继承属性。

### 与服务端渲染一起使用

在使用服务器端渲染时，您可能会注意到在水合后布局发生移动。这是因为触发器执行客户端代码以显示所选项的文本。为了避免这种布局移动，您可以通过映射值手动渲染它。

```jsx live=true
() => {
	const data = {
		apple: "苹果",
		orange: "橘子",
	};
	const [value, setValue] = React.useState("apple");
	return (
		<Select.Root value={value} onValueChange={setValue}>
			<Select.Trigger>{data[value]}</Select.Trigger>
			<Select.Content>
				<Select.Item value="apple">苹果</Select.Item>
				<Select.Item value="orange">橘子</Select.Item>
			</Select.Content>
		</Select.Root>
	);
};
```

### 带图标

您可以通过手动控制子元素来自定义触发器如何渲染值。例如，您可以在所选项的文本旁边渲染一个图标。

```jsx live=true
() => {
	const data = {
		light: { label: "光明", icon: <SunIcon /> },
		dark: { label: "黑暗", icon: <MoonIcon /> },
	};
	const [value, setValue] = React.useState("light");
	return (
		<Flex direction="column" maxWidth="160px">
			<Select.Root value={value} onValueChange={setValue}>
				<Select.Trigger>
					<Flex as="span" align="center" gap="2">
						{data[value].icon}
						{data[value].label}
					</Flex>
				</Select.Trigger>
				<Select.Content position="popper">
					<Select.Item value="light">光明</Select.Item>
					<Select.Item value="dark">黑暗</Select.Item>
				</Select.Content>
			</Select.Root>
		</Flex>
	);
};
```
