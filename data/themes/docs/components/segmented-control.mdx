metaTitle: 分段控制
metaDescription: 用于在不同值或视图之间切换的切换按钮。
sourcePath: components/segmented-control
---

```jsx live=true
<SegmentedControl.Root defaultValue="inbox">
	<SegmentedControl.Item value="inbox">收件箱</SegmentedControl.Item>
	<SegmentedControl.Item value="drafts">草稿箱</SegmentedControl.Item>
	<SegmentedControl.Item value="sent">已发送</SegmentedControl.Item>
</SegmentedControl.Root>
```

## API 参考

该组件继承了 [切换组原语](/primitives/docs/components/toggle-group) 的属性和功能。它支持 [常见的边距属性](/themes/docs/overview/layout#margin-props)。

### 根节点

包含控制的项。

<ThemesPropsTable defs="segmentedControlRootPropDefs" />

### 项

表示控制的单个值。

## 示例
