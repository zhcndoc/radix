## 示例

### 缩略

使用 `truncate` 属性在文本溢出其容器时用省略号缩略文本。

```jsx live=true line=2
<Flex maxWidth="300px">
	<Strong truncate>
		排版的目标是以一种比例关系，结合字体大小、行高和行宽，最大化美感，使阅读变得更容易和愉悦。
	</Strong>
</Flex>
```
