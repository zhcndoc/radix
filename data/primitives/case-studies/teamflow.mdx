# Teamflow

<HeroQuote>
	一旦我们采用了 Radix Primitives，我们看到了明显的性能提升。

**描述一下你正在使用 Radix Primitives 的项目类型？**

Teamflow 是一个一体化的协作中心，远程团队在这里相聚、头脑风暴，完成他们一生中最好的工作。我们结合了 WebGL、WebRTC 和 React，将网页推向极限。这意味着性能对我们至关重要。我们将 Radix 作为我们设计系统的快速且可访问的基础。

<br />

**Radix Primitives 在哪些方面帮助了你和你的公司？**

我们一开始使用一个现成的 UI 库以快速开展工作，但很快遇到了性能瓶颈。一旦我们采用了 Radix Primitives，我们看到了明显的性能提升。我们开始用它替换模态框和提示框，立刻看到了巨大的收益。适应组件到我们设计团队的规范也更快。Radix 使我们能花更多时间专注于核心产品。

<br />

**你对 Radix 开发者体验有何看法？**

无头 UI 模型易于采用。我们从未需要与库斗争以应用我们的自定义样式。Radix 与任何 CSS 层的兼容性很好。我们需要一个零运行时的 CSS 库，并且很快就能够集成 Vanilla Extract。Radix 组件的 API 设计良好且一致，同时易于学习和维护。

<br />

**你对 Radix 文档有何看法？**

文档清晰简洁。"解剖"部分在了解如何使用某个组件时非常有帮助。Github 源代码链接非常棒，当我们想深入研究时可以使用。键盘交互文档详细。作为开发者，你能够理解发生了什么，没有不愉快的惊喜。示例展示了真实的复杂用例，而不仅仅是顺利的路径。
