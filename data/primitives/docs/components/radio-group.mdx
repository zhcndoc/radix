## 安装

从命令行安装组件。

```bash
npm install @radix-ui/react-radio-group
```

## 结构

导入所有部分并将它们组合在一起。

```jsx
import { RadioGroup } from "radix-ui";

export default () => (
    <RadioGroup.Root>
        <RadioGroup.Item>
            <RadioGroup.Indicator />
        </RadioGroup.Item>
    </RadioGroup.Root>
);
```
