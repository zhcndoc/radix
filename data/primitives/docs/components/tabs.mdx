## 安装

从命令行安装组件。

```bash
npm install @radix-ui/react-tabs
```

## 解剖结构

导入所有部分并将它们组合在一起。

```jsx
import { Tabs } from "radix-ui";

export default () => (
	<Tabs.Root>
		<Tabs.List>
			<Tabs.Trigger />
		</Tabs.List>
		<Tabs.Content />
	</Tabs.Root>
);
```

## 示例

### 垂直

您可以通过使用 `orientation` 属性来创建垂直选项卡。

```jsx line=4
import { Tabs } from "radix-ui";

export default () => (
	<Tabs.Root defaultValue="tab1" __orientation__="vertical">
		<Tabs.List aria-label="选项卡示例">
			<Tabs.Trigger value="tab1">一</Tabs.Trigger>
			<Tabs.Trigger value="tab2">二</Tabs.Trigger>
			<Tabs.Trigger value="tab3">三</Tabs.Trigger>
		</Tabs.List>
		<Tabs.Content value="tab1">选项卡一内容</Tabs.Content>
		<Tabs.Content value="tab2">选项卡二内容</Tabs.Content>
		<Tabs.Content value="tab3">选项卡三内容</Tabs.Content>
	</Tabs.Root>
);
```
