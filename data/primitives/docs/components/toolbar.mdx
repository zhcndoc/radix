## 安装

从命令行安装组件。

```bash
npm install @radix-ui/react-toolbar
```

## 解剖结构

导入组件。

```jsx
import { Toolbar } from "radix-ui";

export default () => (
	<Toolbar.Root>
		<Toolbar.Button />
		<Toolbar.Separator />
		<Toolbar.Link />
		<Toolbar.ToggleGroup>
			<Toolbar.ToggleItem />
		</Toolbar.ToggleGroup>
	</Toolbar.Root>
);
```

## 示例

### 与其他原语一起使用

所有暴露 `Trigger` 部分的原语，例如 `Dialog`、`AlertDialog`、`Popover`、`DropdownMenu` 可以通过使用 [`asChild` 属性](/primitives/docs/guides/composition) 在工具条中组合。

以下是使用我们的 `DropdownMenu` 原语的示例。

```jsx line=8-10
import { Toolbar, DropdownMenu } from "radix-ui";

export default () => (
	<Toolbar.Root>
		<Toolbar.Button>操作 1</Toolbar.Button>
		<Toolbar.Separator />
		<DropdownMenu.Root>
			<Toolbar.Button __asChild__>
				<DropdownMenu.Trigger>触发器</DropdownMenu.Trigger>
			</Toolbar.Button>
			<DropdownMenu.Content>…</DropdownMenu.Content>
		</DropdownMenu.Root>
	</Toolbar.Root>
);
```
