## 安装

从命令行安装组件。

```bash
npm install @radix-ui/react-switch
```

## 解剖

导入所有部分并将它们组合在一起。

```jsx
import { Switch } from "radix-ui";

export default () => (
	<Switch.Root>
		<Switch.Thumb />
	</Switch.Root>
);
```
