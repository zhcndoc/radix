## 安装

从命令行安装组件。

```bash
npm install @radix-ui/react-label
```

## 解剖结构

导入组件。

```jsx
import { Label } from "radix-ui";

export default () => <Label.Root />;
```

## 可及性

此组件基于原生 `label` 元素，当包装控件或使用 `htmlFor` 属性时，它将自动应用正确的标记。为了确保自定义控件正常工作，请确保它们使用原生元素，如 `button` 或 `input` 作为基础。
