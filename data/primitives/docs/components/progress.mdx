## 安装

从命令行安装该组件。

```bash
npm install @radix-ui/react-progress
```

### 结构

导入所有部分并将它们组合在一起。

```jsx
import { Progress } from "radix-ui";

export default () => (
	<Progress.Root>
		<Progress.Indicator />
	</Progress.Root>
);
```

## 可及性

遵循 [`progressbar` 角色的要求](https://www.w3.org/WAI/ARIA/apg/patterns/meter)。
