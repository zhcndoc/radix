# 方向提供者

<Description>包装您的应用以提供全局阅读方向。</Description>

<Highlights
	features={["使所有原始组件继承全球阅读方向。"]}
/>

## 安装

从命令行安装组件。

```bash
npm install @radix-ui/react-direction
```

## 结构

导入组件。

```jsx
import { Direction } from "radix-ui";

export default () => <Direction.Provider />;
```

## API 参考

### 提供者

当创建需要从右到左（RTL）阅读方向的本地化应用时，您需要使用 `Direction.Provider` 组件包装您的应用，以确保所有原始组件根据 `dir` 属性调整其行为。

<PropsTable
	data={[
		{
			name: "dir",
			required: false,
			type: '"ltr" | "rtl"',
			typeSimple: "枚举",
			description: (
				<span>
					您应用的全局阅读方向。所有原始组件将继承此设置。
				</span>
			),
		},
	]}
/>

## 示例

使用方向提供者。

```jsx
import { Direction } from "radix-ui";

export default () => (
	<Direction.Provider dir="rtl">{/* 您的应用 */}</Direction.Provider>
);
```
