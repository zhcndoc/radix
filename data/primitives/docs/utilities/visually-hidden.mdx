## 安装

从命令行安装组件。

```bash
npm install @radix-ui/react-visually-hidden
```

## 结构

导入组件。

```jsx
import { VisuallyHidden } from "radix-ui";

export default () => <VisuallyHidden.Root />;
```

## 基本示例

使用视觉隐藏的原语。

```jsx
import { VisuallyHidden } from "radix-ui";
import { GearIcon } from "@radix-ui/react-icons";

export default () => (
	<button>
		<GearIcon />
		<VisuallyHidden.Root>设置</VisuallyHidden.Root>
	</button>
);
```
